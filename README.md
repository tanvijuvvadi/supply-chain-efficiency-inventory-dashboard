# 📦 Supply Chain Efficiency & Inventory Optimization Dashboard

## 📊 Project Overview
This project is a Power BI dashboard designed to analyze supply chain performance, inventory efficiency, and supplier delivery metrics. It helps businesses monitor stock levels, optimize inventory, and improve operational decision-making.

## 🎯 Objective
- Monitor inventory levels across product categories
- Track supplier delivery performance
- Analyze demand trends and sales performance
- Improve supply chain efficiency
- Support data-driven business decisions

## 🛠 Tools Used
- Power BI
- Microsoft Excel (Dataset)
- DAX (Data Analysis Expressions)
- Power Query (Data Cleaning & Transformation)

## 📁 Dataset Description
The dataset includes:
- Product details (Product Name, Category)
- Inventory data (Opening Stock, Closing Stock, Incoming Stock)
- Sales data (Units Sold)
- Supplier details
- Delivery performance (On-Time / Delayed)
- Lead time and cost metrics

## ⚙️ Power Query Steps
- Removed null values
- Changed data types
- Created Month, Quarter, Year columns
- Cleaned and standardized column names
- Handled duplicates
- Formatted numeric and percentage fields

## 📐 Key DAX Measures
- Total Inventory
- Total Units Sold
- Total Incoming Stock
- Total Closing Stock
- On-Time Delivery %
- Average Lead Time
- Inventory Holding Cost

## 📊 Dashboard Features

### 🔹 KPI Cards
- 📦 Total Inventory  
- 📊 Total Units Sold  
- 🚚 On-Time Delivery %  
- ⏳ Average Lead Time  
- 💰 Inventory Holding Cost  
- 📥 Incoming Stock  

### 🔹 Visualizations
- 📦 Inventory by Product Category (Column Chart)
- 🚚 Supplier Delivery Performance (Bar Chart)
- 📈 Monthly Units Sold Trend (Line Chart)
- 🍩 Delivery Status Distribution (Donut Chart)
- 📊 Forecast vs Actual Demand (Clustered Column Chart)
- 📋 Product-Level Inventory Table (Matrix)

### 🔹 Filters / Slicers
- Region  
- Supplier  
- Product Category  
- Warehouse  
- Month 

## 📈 Key Insights
- Identified product categories with high and low inventory
- Evaluated supplier performance based on on-time delivery
- Observed sales trends over time
- Analyzed delivery delays and operational efficiency
- Provided actionable insights to improve supply chain performance

## 🚀 Conclusion
This dashboard provides a comprehensive view of supply chain operations and helps businesses optimize inventory, improve supplier performance, and enhance decision-making.
